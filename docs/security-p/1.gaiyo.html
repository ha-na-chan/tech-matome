1. 情報セキュリティマネジメントについて（概要）¶

1.1. はじめに¶

わが国はもとより、世界中で高度な情報化社会が到来していて、情報資産への脅威が決して小さくないことに疑いはないように思う。しかしその対策は多岐にわたり、何から手を付けて良いかわかりづらい現状にある。本まとめは、情シス的に近い立ち位置の筆者が上の人を動かすための調査事項を個人的にまとめたものである。

1.2. 用語の定義¶

本まとめにおける用語の意義は、次に掲げるところによる。

用語 意味 備考
情報セキュリティ事故 情報の機密性が失われる事態を言う。具体的には、情報の漏洩や、権限の設定ミスによる意図しないアクセスが可能な状態となることなど。 故意のものを含む。
個人情報 生存する個人に関する情報であって、特定の個人を識別できるもの。 個人情報保護法（第2条）
可用性 認可された利用者が、必要なときに情報にアクセスできることを確実にすること。  
完全性 情報および処理方法の正確さおよび完全である状態を安全防護すること。  
機密性 情報にアクセスすることが認可されたものだけがアクセスできることを確実にすること。  
情報セキュリティポリシー 情報の機密性や完全性、可用性を維持していくために規定する組織の方針や行動指針をまとめたもの。  
標的型攻撃 特定の組織を狙って、機密情報や知的財産、アカウント情報（ID、パスワード）などを窃取しようとする攻撃のこと。
